# Fly Right

Live Site: https://fly-right.web.app/

## Features
- User authentication with email/password and Google sign-in.
- Logged-in users can add new visa information through a protected route.
- All visas are displayed in a grid with essential details.
- Each visa has a dedicated details page.
- Users can view and manage the visas they have added.
- Update and delete UI prepared for future use.
- Clean and consistent UI using Tailwind CSS and DaisyUI.


## Technologies Used
- React
- Firebase Authentication
- Express.js (backend)
- MongoDB
- Tailwind CSS
- DaisyUI
